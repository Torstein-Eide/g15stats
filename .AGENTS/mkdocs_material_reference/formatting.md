# Formatting

Reference:
- https://squidfunk.github.io/mkdocs-material/reference/formatting/
- Local: `.tmp/mkdocs-material-reference/formatting.md`

## When to use in this repo
- Keep docs scan-friendly and consistent.
- Emphasize commands/options without over-styling.

## Example for G15Stats docs
```markdown
Use `g15stats -h` to list all options.

**Recommended:** verify with `mkdocs build --strict` before commit.
```
